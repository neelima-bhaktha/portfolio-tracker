# Portfolio Tracker

A professional full stack application for tracking personal investments.

## Overview

Portfolio Tracker allows you to manage your financial assets in one place. You can create multiple portfolios, log buy and sell transactions, and visualize your asset allocation. The application is built with modern web technologies to ensure speed, security, and a beautiful user experience.

## Features

* Secure user authentication with JWT
* Create and manage multiple investment portfolios
* Search for stocks and view market data
* Log your trades and track total invested value
* Beautiful data visualization with interactive charts
* Dark mode UI with modern glassmorphism design

## Technology Stack

### Frontend
* React
* Vite
* Tailwind CSS
* Recharts
* Lucide Icons

### Backend
* Node.js
* Express
* TypeORM
* SQLite Database
* JSON Web Tokens

## Getting Started

Follow these steps to run the project on your local machine.

### Prerequisites

* Node.js installed on your computer
* Git installed on your computer

### Installation

1. Clone the repository to your local machine
2. Open a terminal and navigate to the project folder
3. Open a second terminal window so you have two open

### Backend Setup

1. In the first terminal, navigate to the backend folder
2. Install the required dependencies by running `npm install`
3. Copy the example environment file and rename it to `.env`
4. Start the backend server by running `npm run dev`

### Frontend Setup

1. In the second terminal, navigate to the frontend folder
2. Install the required dependencies by running `npm install`
3. Start the frontend development server by running `npm run dev`
4. Open your browser and visit the URL shown in the terminal
